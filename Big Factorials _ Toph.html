Big Factorials

Limits 1s, 512 MB

Given an integer N, print the trailing 4 digits of N!N!N! (N factorial).

N!=N×(N−1)×(N−2)×...×1N! = N \times (N-1) \times (N-2) \times ... \times 1N!=N×(N−1)×(N−2)×...×1

Here are some examples of N!N!N! and their last 4 digits.

NN Factorial (N!N!N!)Last 4 Digits

Input

The input will contain an integer NNN (0<N<10000 < N < 10000<N<1000).

Output

Print the last 4 digits of N!N!N!. In case N!N!N! has less than 4 digits, do not print any leading zeros.

Examples
Input
4
Output
